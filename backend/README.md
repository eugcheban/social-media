# Social Media App

This is a social media application with a Django backend and a React frontend.

## Prerequisites

- Docker
- Docker Compose

## How to run

1.  Clone the repository
2.  Navigate to the `social-media` directory
3.  Run the following command:

```
docker-compose up
```

This will build the images and start the containers. The backend will be available at `http://localhost:8000` and the frontend at `http://localhost:3000`.
